# HTML
## HTML - 1
### 01. HTML 기본 문서 만들기
#### HTML 이란?
HTML(Hyper Text Markup Language)
- 웹 문서의 구조 및 내용을 정의해 웹 문서를 만드는 언어이다.
- 하이퍼텍스트란? 문서를 서로 연결해 주는 링크를 의미한다.
- 마크럽이란? 웹 브라우저에 내용을 보여주는 텍스트, 이미지, 영상 등의 위치를 표시
  
##### 주요 구성 요소
1. **태그 (Tags)**: &lt;`태그명`&gt; 형식, 웹 페이지의 다양한 요소를 정의
2. **속성 (Attributes)**: 태그에 추가 정보 제공, 예: &lt;`a href="주소"`&gt;
3. **요소 (Elements)**: 시작 태그와 종료 태그 사이의 내용, 예: &lt;`p`&gt;`문단`&lt;`p`&gt;
  
#### 기본 구조
HTML문서는 정해진 형식에 맞춰어 내용을 입력한다.  
&lt;!DOCTYPE html&gt; 시작해서  
&lt;html&gt;,&lt;head&gt;,&lt;body&gt; 3개의 영역으로 구성한다.
  
```html
<!DOCTYPE html>
<html>
  <head>
    <title>페이지 제목</title>
  </head>
  <body>
    <h1>제목</h1>
    <p>문단</p>
  </body>
</html>
```
  
1. &lt;!DOCTYPE html&gt; :  
  - 현재 문서가 HTML 5 언어로 작성한 웹 문서
2. &lt;html&gt; ~ &lt;/html&gt; :  
  - 웹 문서의 시작과 끝을 나타내는 태그
3. &lt;head&gt; ~ &lt;/head&gt; :  
  - 웹 브라우저가 웹 문서를 해석하는 데 필요한 정보를 입력하는 부분
  - &lt;meta&gt; :  
      - 데이터에 관한 데이터
      - 웹 문서와 관련된 정보를 지정
      - 가장 중요한 화면에 글자를 표시할 때 어떤 **인코딩**을 사용할지 지정하는 것
      - 웹 서버는 영어가 기본, 한글 -&gt; **"UTF-8"**  
      ```html
      <meta charset="UTF-8">
      <meta name="keywords" content="html의 구조">
      <meta name="description" content="html의 구조를 알아봅시다">
      <meta name="author" content="Kyunghee Ko">
      ```
  - &lt;title&gt; ~ &lt;/title&gt; :  
      - 웹 문서의 제목을 입력하는 태그로 가장 중요한 태그
      - 제목 표시줄의 제목으로 표시된다.
4. &lt;body&gt; ~ &lt;/body&gt; :  
  - 실제로 웹 브라우저 화면에 나타나는 내용
  
#### 시맨틱 태그
 시맨틱(semantic) 태그는 HTML에서 웹 페이지의 다양한 부분을 명확하고 의미론적으로 정의하는 데 사용되는 태그이다. 이를 사용하여 웹 페이지의 구조가 더 명확해지고, 검색 엔진 최적화(SEO)에 유리하며, 웹 접근성이 향상된다.
  
**주요 시맨틱 태그**
- &lt;header&gt;,&lt;nav&gt;,&lt;main&gt;,&lt;footer&gt; 가장 베이스적인 
1. &lt;header&gt;: 페이지나 섹션의 머리말을 나타냅니다. 보통 로고, 제목, 내비게이션 메뉴 등을 포함합니다.
2. &lt;nav&gt;: 내비게이션 링크를 위한 섹션을 정의합니다. 주로 메뉴, 목차, 인덱스 등에 사용됩니다.
3. &lt;main&gt;: 웹 페이지의 주요 콘텐츠를 나타냅니다. &lt;main&gt; 요소는 문서당 한 번만 사용해야 합니다.
4. &lt;section&gt;: 문서의 독립적인 섹션을 정의하며, 주로 관련 콘텐츠를 그룹화하는 데 사용됩니다.
5. &lt;article&gt;: 독립적으로 구분되거나 재사용 가능한 섹션을 나타냅니다. 예를 들어 포스트, 뉴스 기사, 블로그 항목 등이 여기에 해당합니다.
6. &lt;aside&gt;: 문서의 주 내용과 간접적으로만 관련된 부분을 나타냅니다. 사이드바나 콜아웃 박스 등에 사용됩니다.
7. &lt;footer&gt;: 페이지나 섹션의 바닥글을 나타냅니다. 일반적으로 저자 정보, 저작권 정보, 관련 문서 링크 등이 포함됩니다.
8. &lt;figure&gt;와 &lt;figcaption&gt;: 독립적인 콘텐츠(예: 이미지, 다이어그램, 코드 스니펫 등)를 나타내며, &lt;figcaption&gt;은 해당 콘텐츠의 설명이나 제목을 제공합니다.
9. &lt;div&gt;: 헤더나 내비게이션, 푸터 등을 구별할 때 &lt;div&gt; 태그를 사용한다.
  


## HTML - 2
### 02. 웹 문서에 다양한 내용 입력하기
#### 텍스트 입력
1. &lt;hn&gt; 태그
웹 문서에서 제목은 다른 텍스트보다 크고 진하게 표시한다.  
```html
<h1>제목</h1>
```
형태로 **&lt;hn&gt;제목&lt;/hn&gt;**의 형태로 n = 1 ~ 6 까지 존재한다. 
   
2. 단락 : &lt;P&gt; 태그 / 줄 바꿈 &lt;br&gt; 태그
```html
<p>내용</p>
<br>
```
**&lt;p&gt;~&lt;/p&gt;**   
  - 웹 문서에서 텍스트 단락을 입력할 때 많이 사용하는 태그이다.  
  - 태그 안에서 줄을 바꾸더라도 웹 브라우저에서는 한 줄로 표시된다.
**&lt;br&gt;**
  - 원하는 위치에서 줄을 바꿀때 사용하는 태그이다.
  - 닫는 태그가 필요 없다.
  - &lt;br&gt; 두번 하면 &lt;p&gt;를 새로 만든것과 같은 간격이 생김 하지만, CSS에서 문제 발생
  
3. 인용 &lt;blockquote&gt; 태그
```html
<blockquote>인용문</blockquote>
```
브라우저가 인용문을 인식할 수 있게 하는 태그이다.  
&lt;blockquote&gt; 태그 안의 내요을 인용문으로 알고 다른 텍스트보다 약간 들여 쓴다.

4. 굵은 표시 &lt;strong&gt;, &lt;b&gt; 태그
```html
<strong>굵게 강조할 텍스트</strong>
<b>굵게 강조할 텍스트</b>
```
&lt;strong&gt;와 &lt;b&gt;는 눈으로 볼 때 차이점이 없으나 화면 낭독기의 기능의 차이가 존재한다.  

- &lt;strong&gt; : 경고나 주의 사항과 같은 중요한 내용
- &lt;b&gt; : 단순히 글자만 굵게 표시할 내용
- &lt;strong&gt;과 &lt;b&gt; 은 줄바꿈을 기본적으로 적용되지 않는다.
  
5. 기울인 텍스트 &lt;em&gt;, &lt;i&gt; 태그
```html
<em>이탤릭체로 강조할 텍스트</em>
<i>이탤릭체로 강조할 텍스트</i>
``` 
- &lt;em&gt; : emphasis의 줄임말, 흐름상 특정 부분 강조할 때 사용
- &lt;i&gt; : italic 줄임말, 용어-관용구 등 다른 텍스트와 구별하기 위해 사용
- &lt;em&gt;과 &lt;i&gt;은 줄바꿈을 기본적으로 적용되지 않는다.

6. 다양한 텍스트 관련 태그
- &lt;abbr&gt;&lt;/abbr&gt; : 줄임말을 표시하고 title 속성을 함께 사용할 수 있다.
- &lt;cite&gt;&lt;/cite&gt; : 웹 문서나 포스트에서 참고 내용을 표시합니다.
- &lt;code&gt;&lt;/code&gt; : 컴퓨터 인식을 위한 소스 코드입니다.
- &lt;small&gt;&lt;/small&gt; : 부가 정보처럼 작게 표시해도 되는 텍스트
- &lt;sub&gt;&lt;/sub&gt; : 아래 첨자를 표시합니다.
- &lt;sup&gt;&lt;/sup&gt; : 위의 첨자를 표시합니다.
- &lt;s&gt;&lt;/s&gt; : 취소선을 표시합니다.
- &lt;u&gt;&lt;/u&gt; : 밑줄을 표시합니다.
- &lt;ins&gt;&lt;/ins&gt; : 공동 작업 문서에서 새로운 내용을 삽입합니다. -&gt; 밑줄이 생김
- &lt;del&gt;&lt;/del&gt; : 공동 작업 문서에서 기존 내용을 삭제합니다. -&gt; 취소선이 생김
  
  
### 목록
1. 순서 있는 목록 &lt;ol&gt;, &lt;li&gt; 태그
```html
<ol>
  <li>항목1</li>
  <li>항목2</li>
</ol>
```
- 순서 있는 목록이란?
  각 항목을 순서대로 나열한 것이다.
- &lt;ol&gt; : ordered list, 순서 있는 목록은 기본적으로 숫자 1, 2, 3 ~ 으로 번호를 줌
  -  type = "1" : 숫자(기본값)
  -  type = "a" : 영문 소문자
  -  type = "A" : 영문 대문자
  -  type = "i" : 로마 숫자 소문자
  -  type = "I" : 로마 숫자 대문자
  -  start = "n" : n = 시작할 순서
- &lt;li&gt; : list, 리스트의 항목을 넣는 테그
  
2. 순서 없는 목록 &lt;ul&gt;,&lt;li&gt; 태그
```html
<ul>
  <li>항목1</li>
  <li>항목2</li>
</ul>
```
- &lt;ul&gt; : unordered list, 항목의 순서가 중요하지 않을 때 사용

3. 설명 목록 &lt;dl&gt;, &lt;dt&gt;, &lt;dd&gt; 태그
```html
<dl>
 <dt>이름</dt>
 <dd>값</dd>
</dl>
```
설명 목록description list이란 이름name 값 value 형태로 된 목록
- &lt;dl&gt; : &lt;dt&gt;와 &lt;dd&gt;을 넣는 태그
- &lt;dt&gt; : 이름을 넣는 태그
- &lt;dd&gt; : 값을 넣는 태그
&lt;dt&gt;와 &lt;dd&gt;는 한쌍으로 사용한다.
&lt;dt&gt; 태그 하나에 &lt;dd&gt; 태그를 여러개 사용할 수도 있다.
  

## HTML - 3
### 02. 웹 문서에 다양한 내용 입력하기
#### 표
1. 표를 만드는 &lt;table&gt;, &lt;caption&gt; 태그
```html
<table>
  <caption>표 제목</caption>
</table>
```
- &lt;table&gt; : 표의 시작과 끝을 알려주는 태그
- &lt;caption&gt; : 표의 제목으로 표의 위쪽 중앙에 표시

2. 행 &lt;tr&gt;, 태그와 셀 &lt;td&gt;,&lt;th&gt;
```html
<table>
  <tr>
    <td>1행 1열</td>
    <td>1행 2열</td>
  </tr>
  <tr>
    <td>2행 1열</td>
    <td>2행 2열</td>
  </tr>
</table>
```
- &lt;tr&gt; : 행(세로)을 만드는 태그로 이안에 &lt;td&gt;가 포함되어 있다.
- &lt;td&gt; : 열(가로)을 만드는 태그
- &lt;th&gt; : 열(가로)제목행을 만드는 태그로 &lt;td&gt; 대신 사용된다.
  
3. 표의 구조 지정 &lt;thead&gt;, &lt;tbody&gt;, &lt;tfoot&gt; 태그
- &lt;thead&gt; : 표의 제목을 지정하는 태그
- &lt;tbody&gt; : 표의 본문, 데이터가 있는 위치를 지정하는 태그
- &lt;tfoot&gt; : 표의 요양을 지정하는 태그
이러한 표의 구조 지정은 css나 자바스크린트를 적용할 떄 각각 다른 스타일을 적용하거나 기능을 고정하는 등 할 수 있게 해줍니다.
  
4. rowspan, colspan 속성
```html
<td rowspan="합칠 셀의 개수">셀내용</td>
<td colspan="합칠 셀의 개수">셀내용</td>
```
- rowspan : 행(세로) 합치기
- colspan ; 열(가로) 합치기
  
5. 열을 묶어 주는 &lt;col&gt;, &lt;colgroup&gt;
```html
<colgroup>
  <col>
</colgroup>
```
- &lt;colgroup&gt; : 2개 이상의 &lt;col&gt; 묶어서 사용
- &lt;col&gt; : 열 1개에 지정 사용, 표의 열 개수 만큼 넣어야 한다.
    

#### 이미지 삽입
1. 이미지 삽입 &lt;img&gt; 태그
```html
<img src="이미지 파일 경로" alt="대체용 텍스트">
```
- &lt;img&gt; : 웹 문서에 이미지를 삽입할 떄 가장 기본인 태그이다.
  - src 속성 : 
    - 이미지 파일의 경로를 지정
    - 같은 폴더에 있다면 이미지 파일 이름만 적용 가능
    - "../" : 경로를 뒤로 이동 할때 사용
  - alt 속성 : 
    - 화면 낭독기 등에서 이미지를 대신해서 읽어 줄 텍스트를 입력
    - 인터넷인 불안정하거나 이미지 파일 경로를 잘못 넣었을 때 표시됨
  - width 속성 : 이미지의 너비를 조정
  - height 속성 : 이미지의 높이를 조정

  
##### 이미지 종류
  - GIF (Graphics Interchange Format)
    - **사용**: 주로 간단한 애니메이션과 로우 컬러 이미지에 사용됩니다.
    - **색상**: 최대 256색을 지원합니다.
    - **투명도**: 간단한 투명도 지원(완전히 투명하거나 완전히 불투명).
    - **압축**: 손실 없는 압축을 사용합니다.
  - JPG/JPEG (Joint Photographic Experts Group)
    - **사용**: 사진 및 현실적인 이미지에 주로 사용됩니다.
    - **색상**: 높은 색상 깊이를 지원하여 사실적인 이미지를 처리할 수 있습니다.
    - **투명도**: 투명도를 지원하지 않습니다.
    - **압축**: 손실 압축을 사용합니다. 압축률이 높을수록 이미지 품질이 저하될 수 있습니다.
  - PNG (Portable Network Graphics)
    - **사용**: 웹 이미지에 널리 사용되며, 투명도와 고품질 이미지를 필요로 하는 경우에 적합합니다.
    - **색상**: 다양한 색상 모드를 지원하며, GIF보다 더 많은 색상을 표현할 수 있습니다.
    - **투명도**: 알파 채널을 통한 투명도 지원으로, 이미지의 특정 부분을 다양한 수준의 투명도로 표현할 수 있습니다.
    - **압축**: 손실 없는 압축을 사용합니다.
##### px (픽셀)
- **정의**: 디지털 이미지의 기본 단위로, 스크린에 표시되는 이미지의 가장 작은 점.
- **사용**: 이미지, 요소, 레이아웃의 크기를 정확하게 지정하는 데 사용됨. 예: `width: 200px`.
- **특징**:
  - 절대적인 크기 단위, 사용자의 디스플레이 해상도나 브라우저 확대/축소에 영향을 받지 않음.
  - 고정된 크기로 인해 반응형 디자인에서 유연성이 떨어질 수 있음.

##### % (퍼센트)
- **정의**: 백분율 단위, 상위 요소(부모 요소)의 크기에 대한 상대적인 비율로 사용됨.
- **사용**: 요소의 크기를 상위 요소의 크기에 비례하게 조정할 때 사용됨. 예: `width: 50%`.
- **특징**:
  - 상대적 크기 단위, 요소의 크기가 부모 요소의 크기에 따라 변함.
  - 반응형 웹 디자인에 적합, 다양한 화면 크기와 장치에 유연하게 적응함.

#### 오디오와 비디오 삽입
1. 다양한 멀티미디어 파일 삽입 &lt;object&gt;, &lt;embed&gt; 태그
```html
<object width="너비" height="높이" data="파일"></object>
```
- &lt;object&gt; : 오디오, 비디오, 자바애플릿, PDF 등 다양한 멀티미디어 파일을 삽입가능
  - width : 너비를 설정
  - height : 높이를 설정
```html
<embed src="파일경로" width="너비" height="높이">
```
- &lt;embed&gt; : 오디오, 비디오, 이미지 등 다양한 멀티미니더 파일을 삽입할 수 있다. &lt;audio&gt;, &lt;video&gt;, &lt;object&gt;가 지원하지 않는 웹 브라우저에서 사용
  
2. 멀티미디어 파일의 종류
- MP4
  - 다양한 멀티미디어 콘텐츠(비디오, 오디오, 자막, 2D 및 3D 그래픽)를 지원.
  - 고품질 비디오를 낮은 비트레이트로 제공, 인터넷 스트리밍에 적합.
  - 광범위한 장치 및 플랫폼에서 널리 지원됨.
- WebM
  - 고품질의 비디오를 웹에서 스트리밍하기 위해 개발됨.
  - VP8 또는 VP9 비디오 코덱과 Vorbis 또는 Opus 오디오 코덱을 사용.
  - 오픈소스이며, HTML5 비디오 표준의 일부로 지원됨.
  
- MP3
  - 손실 압축 오디오 형식으로, 음악 파일의 크기를 상당히 줄임.
  - CD 품질의 오디오를 제공하면서도 파일 크기를 작게 유지.
  - 가장 널리 사용되는 디지털 오디오 형식 중 하나.
- M4A
  - MPEG-4 Part 14 표준의 일부로 오디오 파일에 사용됨.
  - AAC 또는 ALAC 오디오 코덱을 사용하여 고품질 오디오 제공.
  - MP3에 비해 더 나은 음질을 가지면서 파일 크기는 비슷하거나 작음.
  
3. 오디오와 비디오 파일 삽입 &lt;audio&gt;, &lt;video&gt;
```html
<audio src="오디오 파일 경로" ></audio>
<video src="비디오 파일 경로" ></video>
```
- &lt;audio&gt; : 오디오를 삽입함
- &lt;video&gt; : 비디오를 삽입함

- 속성
  - controls : 컨트롤 바를 표시
  - autoplay : 자동 실행
  - loop : 반복 재생
  - muted : 소리를 제거
  - preload : 파일을 로딩할 것을 지정("auto(기본값)","metadata","none")
  - width, height : 너비와 높이를 지정
  - poster="파일이름"

## HTML - 4
### 03. 입력 양식 작성하기
**입력 양식이란?**
- 특정 항목에 사용자가 뭔가를 입력할 수 있게 만든 것으로 "폼"이라고 한다.
- 예 : 로그인 창, 검색 창
#### 폼 삽입하기
1. 웹에서 만나는 폼  
 우리는 웹에서 `from`을 자주 본다. 아이디와 비밀번호를 입력하는 로그린 화면, 회원 가입 등 사용자가 웹 사이트로 정보를 보낼 수 있는 요소 등이 그 예시이다.
- 동작 원리  
  1. 사용자 정보 입력 후 버튼 클릭
  2. 웹 서로로 전송
  3. 데이터베이스에서 입력 정보 대조
  4. 결과를 사용자에서 전성  
폼 형태는 HTML 태그로 만들지만, 폼에 입력한 사용자 정보는 ASP, PHP, JSP 같은 서버 프로그램밍으로 처리된다.

2. 폼을 만드는 &lt;form&gt; 태그
```html
<form [속성="속성값"]> 여러 폼 요소 </form> 
```
- &lt;form&gt; ~ &lt;/form&gt;  
  이 사이에 여러 가지의 폼 요소를 넣을 수 있다.
- &lt;form&gt; 속성
  - method : 사용자가 입력한 내용을 서버 쪽 프로그램으로 어떻게 넘겨줄 것인지 지정
    - get : 256 ~ 4,096byte까지 서버 전달, 사용자 내용이 그대로 드러나는 단점
    - post : 입력한 내용의 길이에 제한받지 않고 사용자가 입력한 내용 드러나지 않음
  - name : 자바스크립트로 폼을 제어할 때 사용할 폼의 이름을 지정
  - action : 태그 안의 내용을 처리해 줄 서버 프로그램 지정
  - target : action 속성에서 지정한 스크립트 파일을 현재 창이 아닌 다른 위치에서 열도록함
  - autocomplete 속성 : (= "on" & "off")자동 완성ㅇ 기능으로 한두 글짜 정도 입력하면 예전에 입력 내용을 자동으로 보여줌
  
3. 폼 요소를 그룹으로 묶는 &lt;fieldset&gt;, &lt;legend&gt;
```html
<!-- fieldset 기본 구조 -->
<fieldset [속성="속성값"]></fieldset>

<!-- legend 기본 구조 -->
<fieldset>
  <legend></legend>
</fieldset>
```
- &lt;fieldset&gt; :  
  하나의 폼 안에서 여러 구역을 나누어 표시할때 사용한다.
- &lt;legend&gt; :  
  &lt;fieldset&gt; 태그로 묶은 그룹에 제목을 붙인다.
  
4. 폼 요소에 레이블을 붙이는 &lt;label&gt;
```html
<!-- label&input 방법 1 -->
<label>레이블명<input></label>

<!-- label&input 방법 2 -->
<label for="id명">레이블명</label>
<input type="text" id="id명">
```
- &lt;label&gt; : &lt;input&gt; 태그와 같은 폼 요소에 레이블을 붙일 때 사용한다.
  - 방법 1 : 태그 안에 폼 요소를 넣는 방법, 간단하다.
  - 방법 2 : 태그 밖에 따로 사용하는 법, 복잡하지만 떨어져 있어도 둘 사이를 쉽게 연결할 수 있다.
  
  
#### 사용자 입력을 위한 input 태그
##### &lt;input&gt;
  &lt;input&gt; 태그는 다양한 폼에서 사용자가 입력한 정보를 받을 때 사용한다.
  
##### &lt;input&gt; 속성
1. 텍스트 입력
- text : 한 줄짜리 텍스트를 입력할 수 있는 텍스트 박스
- password : 비밀번호를 입력할 수 있는 필드
  - 속성
    - size : 텍스트와 비밀번호 필드의 길이를 지정
    - value : 텍스트 필드 요소가 화면에 표시될 때 텍스트 필드 부분에 보여 주는 내용(비밀번호는 사용 X)
    - maxlength : 텍스트 필드와 비밀번호 필드에 최대 문자 수를 지정
- url : 검색할 때 입력하는 필드
- email : 이메일 주소를 입력하는 필드
- tel : 전화번호를 입력하는 필드
  
2. 체크 박스와 라디오 번튼
- checkbox : 주어진 여러 항목에서 2개 이상 선택할 수 있는 체크 박스
- radio : 주어진 여러 항목에서 1개 선택할 수 있는 라디오 버튼
  - 속성
    - value : 선택한 체크 박스나 라디오 버튼을 서버에게 알려 줄 때 넘겨줄 값을 지정, 이 값은 영문이거나 숫자여야 하며 필수 속성
    - checked : 체크 박스나 라디오 버튼의 항목은 처음에 아무것도 선택되지 않은 상태로 화면에 표시되는데, 여러 항목 중에서 기본으로 선택해 놓고 싶은 항목에 사용
  
3. 숫자 입력
- number : 숫자를 조절할 수 있는 스핀 박스
- range : 숫자를 조절할 수 있는 슬라이드 박스
  - 속성
    - min : 필드에 입력할 수 있는 최솟값을 지정 (기본값 = 0)
    - max : 필드에 입력할 수 있는 최댓값을 지정 (기본값 = 100)
    - step : 숫자 간격을 지정 (기본값 = 1)
    - value : 필드에 표시할 초깃값
  
4. 날짜와 시간 입력
- date : 사용자 지역을 기준으로 날짜(연, 월, 일) yyyy-mm-dd
- month : 사용자 지역을 기준으로 날짜(연, 월) yyyy-mm
- week : 사용자 지역을 기준으로 날짜(연, 주) 몇 번쨰 주
- time : 사용자 지역을 기준으로 시간(시, 분, 초, 분할 초)
- datetime : 국제 표준시(UTC)로 설정된 날짜와 시간(연, 월, 일, 시, 분, 초, 분할 초)
- datetime-local : 사용자가 있는 지역을 기준으로 날짜와 시간(연, 월, 시, 분, 초, 분할 초)
  - 속성
    - min : 날짜나 시간의 범위를 제한할 때 사용, 시작 날짜
    - max : 마지막 날짜
    - step : 스핀 박스의 화살표를 클릭했을 때 증감시킬 크기
    - value : 기본적으로 표시할 날짜나 시간을 지정
  
5. 전송, 리셋 버튼
```html
<input type="submit | reset" value="버튼에 표시할 내용">
```
- submit : 전송 버튼
- reset : 리셋 버튼
  
6. 이미지 버튼
```html
<input type="image" src="이미지 경로" alt="대체 텍스트">
```
- image : submit 버튼 대신 사용할 이미지를 넣는다.
  
7. 기본 버튼
```html
<input type="button" value="버튼에 표시할 내용">
```
- button : 일반 버튼을 넣는다.
  
8. 파일첨부
- file : 파일를 첨부할 수 있는 버튼
  
9. 히든 필드
```html
<input type="hidden" name="이름" value="서버로 넘길 값">
```
- hidden : 사용자에게는 보이지 않지만  서버로 넘겨주는 값이 있는 필드
  
  
## HTML - 5
#### input 태그의 주요 속성
단순히 내용을 입력할 뿐만 아니라 입력란에 커서나 힌트를 표시할 수도 있고, 꼭 입력해야 하는 필드도 지정할 수 있다.
##### autofocus 속성
`autofocus`속성을 사용하면 페이지를 불러오자마자 폼에서 원하는 요소에 마우스 포인터를 표시할 수 있다.
```html
<input type="텍스트-입력-필드" autofocus required>
``` 
  
##### 힌트를 표시해 주는 placeholder 속성
사용자가 텍스트를 입력할 때 입력란에 적당한 힌트 내용을 표시해서 그 필드를 클릭하면 내용이 사라지도록 만들 수 있다. 이러한 힌트 표시할 때 `placeholder` 속성을 사용한다.

##### 읽기 전용 필드 readonly 속성
```html
<input type=텍스트-입력-필드 readonly>
```
`readonly`속성은 사용자가 입력하지는 못하고 읽기 전용 필드를 만들 수 있다.(readonly="readonly | turn")
  
##### 필수 입력 required 속성
submit 버튼으로 폼을 서버로 전송하는데, 필수 필드에 내용이 모두 채워졌는지 검사해야한다. 이때 `required` 속성을 사용한다.
  
#### 폼에서 사용하는 여러 가지 태그
##### 여러 줄 입력 &lt;textarea&gt;
```html
<textarea>내용</textarea>
```
textarea 태그는 텍스트를 여러 줄 입력하는 영역을 만들 수도 있다. 이 영역을 텍스트 영역이라고 한다.
- 속성
  - cols : 텍스트 영역의 가로 너비를 문자 단위로 지정
  - rows : 텍스트 영역의 세로 길이를 줄 단위로 지정, 지정한 숫자보다 줄 개수가 많아지면 스크롤 막개가 생김
  
##### 드롭다운 목록 &lt;select&gt;, &lt;option&gt;
```html
<select>
  <option value="값1">내용1</option>
  <option value="값2">내용2</option>
  ...
</select>
```
사용자가 내용을 직접입력하지 않고 여러 옵션 중에서 선택하게 하려면 드롭다운 목록이나 데이터 목록을 사용한다. 
- &lt;select&gt; : 목록의 시작과 끝을 표시
  - size : 화면에 표시할 드롭다운 항목의 개수를 지정
  - multiple : 드롭다운 목록에서 둘 이상의 항목을 선택할 때 사용
- &lt;option&gt; : 태그를 사용해 원하는 항목을 추가, value 속성으로 값 지정
  - value : 서버로 넘겨줄 값을 지정
  - selected : 드롭다운 메뉴를 삽입할 때 기본적으로 선택해서 보여 줄 항목 지정
    
##### 데이터 목록 만들어 주는 &lt;datalist&gt;, &lt;option&gt;
```html
<input for="text" list="데이터 목록 id">
<datalist>
  <option value="서버로 넘길 값1">내용1</option>
  <option value="서버로 넘길 값2">내용2</option>
</datalist>
```
데이터 목록을 사용하면 텍스트 필드에 값을 직접 입력하지 않고 미리 만들어 놓은 값 중에서 선택
- &lt;datalist&gt; : 데이터 목록의 시작과 끝
  
##### &lt;button&gt; 태그
```html
<button type="submit">내용</button>
<button type="reset">내용</button>
<button type="button">내용</button>
```
