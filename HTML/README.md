# HTML
## HTML - 1
### 01. HTML 기본 문서 만들기
#### HTML 이란?
HTML(Hyper Text Markup Language)
- 웹 문서의 구조 및 내용을 정의해 웹 문서를 만드는 언어이다.
- 하이퍼텍스트란? 문서를 서로 연결해 주는 링크를 의미한다.
- 마크럽이란? 웹 브라우저에 내용을 보여주는 텍스트, 이미지, 영상 등의 위치를 표시
  
##### 주요 구성 요소
1. **태그 (Tags)**: `<태그명>` 형식, 웹 페이지의 다양한 요소를 정의
2. **속성 (Attributes)**: 태그에 추가 정보 제공, 예: `<a href="주소">`
3. **요소 (Elements)**: 시작 태그와 종료 태그 사이의 내용, 예: `<p>문단</p>`
  
#### 기본 구조
HTML문서는 정해진 형식에 맞춰어 내용을 입력한다.  
<!DOCTYPE html> 시작해서  
<html>,<head>,<body> 3개의 영역으로 구성한다.
  
```html
<!DOCTYPE html>
<html>
  <head>
    <title>페이지 제목</title>
  </head>
  <body>
    <h1>제목</h1>
    <p>문단</p>
  </body>
</html>
```
  
1. <!DOCTYPE html> :  
  - 현재 문서가 HTML 5 언어로 작성한 웹 문서
2. <html> ~ </html> :  
  - 웹 문서의 시작과 끝을 나타내는 태그
3. <head> ~ </head> :  
  - 웹 브라우저가 웹 문서를 해석하는 데 필요한 정보를 입력하는 부분
  - <meta> :  
      - 데이터에 관한 데이터
      - 웹 문서와 관련된 정보를 지정
      - 가장 중요한 화면에 글자를 표시할 때 어떤 **인코딩**을 사용할지 지정하는 것
      - 웹 서버는 영어가 기본, 한글 -> **"UTF-8"**  
      ```html
      <meta charset="UTF-8">
      <meta name="keywords" content="html의 구조">
      <meta name="description" content="html의 구조를 알아봅시다">
      <meta name="author" content="Kyunghee Ko">
      ```
  - <title> ~ </title> :  
      - 웹 문서의 제목을 입력하는 태그로 가장 중요한 태그
      - 제목 표시줄의 제목으로 표시된다.
4. <body> ~ </body> :  
  - 실제로 웹 브라우저 화면에 나타나는 내용
  
#### 시맨틱 태그
 시맨틱(semantic) 태그는 HTML에서 웹 페이지의 다양한 부분을 명확하고 의미론적으로 정의하는 데 사용되는 태그이다. 이를 사용하여 웹 페이지의 구조가 더 명확해지고, 검색 엔진 최적화(SEO)에 유리하며, 웹 접근성이 향상된다.
  
**주요 시맨틱 태그**
- <header>,<nav>,<main>,<footer> 가장 베이스적인 
1. <header>: 페이지나 섹션의 머리말을 나타냅니다. 보통 로고, 제목, 내비게이션 메뉴 등을 포함합니다.
2. <nav>: 내비게이션 링크를 위한 섹션을 정의합니다. 주로 메뉴, 목차, 인덱스 등에 사용됩니다.
3. <main>: 웹 페이지의 주요 콘텐츠를 나타냅니다. <main> 요소는 문서당 한 번만 사용해야 합니다.
4. <section>: 문서의 독립적인 섹션을 정의하며, 주로 관련 콘텐츠를 그룹화하는 데 사용됩니다.
5. <article>: 독립적으로 구분되거나 재사용 가능한 섹션을 나타냅니다. 예를 들어 포스트, 뉴스 기사, 블로그 항목 등이 여기에 해당합니다.
6. <aside>: 문서의 주 내용과 간접적으로만 관련된 부분을 나타냅니다. 사이드바나 콜아웃 박스 등에 사용됩니다.
7. <footer>: 페이지나 섹션의 바닥글을 나타냅니다. 일반적으로 저자 정보, 저작권 정보, 관련 문서 링크 등이 포함됩니다.
8. <figure>와 <figcaption>: 독립적인 콘텐츠(예: 이미지, 다이어그램, 코드 스니펫 등)를 나타내며, <figcaption>은 해당 콘텐츠의 설명이나 제목을 제공합니다.
9. <div>: 헤더나 내비게이션, 푸터 등을 구별할 때 <div> 태그를 사용한다.
  


## HTML - 2
### 02. 웹 문서에 다양한 내용 입력하기
#### 텍스트 입력
1. <hn> 태그
웹 문서에서 제목은 다른 텍스트보다 크고 진하게 표시한다.  
```html
<h1>제목</h1>
```
형태로 **<hn>제목</hn>**의 형태로 n = 1 ~ 6 까지 존재한다. 
   
2. 단락 : <P> 태그 / 줄 바꿈 <br> 태그
```html
<p>내용</p>
<br>
```
**<p>~</p>**   
  - 웹 문서에서 텍스트 단락을 입력할 때 많이 사용하는 태그이다.  
  - 태그 안에서 줄을 바꾸더라도 웹 브라우저에서는 한 줄로 표시된다.
**<br>**
  - 원하는 위치에서 줄을 바꿀때 사용하는 태그이다.
  - 닫는 태그가 필요 없다.
  - <br> 두번 하면 <p>를 새로 만든것과 같은 간격이 생김 하지만, CSS에서 문제 발생
  
3. 인용 <blockquote> 태그
```html
<blockquote>인용문</blockquote>
```
브라우저가 인용문을 인식할 수 있게 하는 태그이다.  
<blockquote> 태그 안의 내요을 인용문으로 알고 다른 텍스트보다 약간 들여 쓴다.

4. 굵은 표시 <strong>, <b> 태그
```html
<strong>굵게 강조할 텍스트</strong>
<b>굵게 강조할 텍스트</b>
```
<strong>와 <b>는 눈으로 볼 때 차이점이 없으나 화면 낭독기의 기능의 차이가 존재한다.  

- <strong> : 경고나 주의 사항과 같은 중요한 내용
- <b> : 단순히 글자만 굵게 표시할 내용
- <strong>과 <b> 은 줄바꿈을 기본적으로 적용되지 않는다.
  
5. 기울인 텍스트 <em>, <i> 태그
```html
<em>이탤릭체로 강조할 텍스트</em>
<i>이탤릭체로 강조할 텍스트</i>
``` 
- <em> : emphasis의 줄임말, 흐름상 특정 부분 강조할 때 사용
- <i> : italic 줄임말, 용어-관용구 등 다른 텍스트와 구별하기 위해 사용
- <em>과 <i>은 줄바꿈을 기본적으로 적용되지 않는다.

6. 다양한 텍스트 관련 태그
- <abbr></abbr> : 줄임말을 표시하고 title 속성을 함께 사용할 수 있다.
- <cite></cite> : 웹 문서나 포스트에서 참고 내용을 표시합니다.
- <code></code> : 컴퓨터 인식을 위한 소스 코드입니다.
- <small></small> : 부가 정보처럼 작게 표시해도 되는 텍스트
- <sub></sub> : 아래 첨자를 표시합니다.
- <sup></sup> : 위의 첨자를 표시합니다.
- <s></s> : 취소선을 표시합니다.
- <u></u> : 밑줄을 표시합니다.
- <ins></ins> : 공동 작업 문서에서 새로운 내용을 삽입합니다. -> 밑줄이 생김
- <del></del> : 공동 작업 문서에서 기존 내용을 삭제합니다. -> 취소선이 생김
  
  
### 목록
1. 순서 있는 목록 <ol>, <li> 태그
```html
<ol>
  <li>항목1</li>
  <li>항목2</li>
</ol>
```
- 순서 있는 목록이란?
  각 항목을 순서대로 나열한 것이다.
- <ol> : ordered list, 순서 있는 목록은 기본적으로 숫자 1, 2, 3 ~ 으로 번호를 줌
  -  type = "1" : 숫자(기본값)
  -  type = "a" : 영문 소문자
  -  type = "A" : 영문 대문자
  -  type = "i" : 로마 숫자 소문자
  -  type = "I" : 로마 숫자 대문자
  -  start = "n" : n = 시작할 순서
- <li> : list, 리스트의 항목을 넣는 테그
  
2. 순서 없는 목록 <ul>,<li> 태그
```html
<ul>
  <li>항목1</li>
  <li>항목2</li>
</ul>
```
- <ul> : unordered list, 항목의 순서가 중요하지 않을 때 사용

3. 설명 목록 <dl>, <dt>, <dd> 태그
```html
<dl>
 <dt>이름</dt>
 <dd>값</dd>
</dl>
```
설명 목록description list이란 이름name 값 value 형태로 된 목록
- <dl> : <dt>와 <dd>을 넣는 태그
- <dt> : 이름을 넣는 태그
- <dd> : 값을 넣는 태그
<dt>와 <dd>는 한쌍으로 사용한다.
<dt> 태그 하나에 <dd> 태그를 여러개 사용할 수도 있다.
  

## HTML - 3
### 02. 웹 문서에 다양한 내용 입력하기
#### 표
1. 표를 만드는 <table>, <caption> 태그
```html
<table>
  <caption>표 제목</caption>
</table>
```
- <table> : 표의 시작과 끝을 알려주는 태그
- <caption> : 표의 제목으로 표의 위쪽 중앙에 표시

2. 행 <tr>, 태그와 셀 <td>,<th>
```html
<table>
  <tr>
    <td>1행 1열</td>
    <td>1행 2열</td>
  </tr>
  <tr>
    <td>2행 1열</td>
    <td>2행 2열</td>
  </tr>
</table>
```
- <tr> : 행(세로)을 만드는 태그로 이안에 <td>가 포함되어 있다.
- <td> : 열(가로)을 만드는 태그
- <th> : 열(가로)제목행을 만드는 태그로 <td> 대신 사용된다.
  
3. 표의 구조 지정 <thead>, <tbody>, <tfoot> 태그
- <thead> : 표의 제목을 지정하는 태그
- <tbody> : 표의 본문, 데이터가 있는 위치를 지정하는 태그
- <tfoot> : 표의 요양을 지정하는 태그
이러한 표의 구조 지정은 css나 자바스크린트를 적용할 떄 각각 다른 스타일을 적용하거나 기능을 고정하는 등 할 수 있게 해줍니다.
  
4. rowspan, colspan 속성
```html
<td rowspan="합칠 셀의 개수">셀내용</td>
<td colspan="합칠 셀의 개수">셀내용</td>
```
- rowspan : 행(세로) 합치기
- colspan ; 열(가로) 합치기
  
5. 열을 묶어 주는 <col>, <colgroup>
```html
<colgroup>
  <col>
</colgroup>
```
- <colgroup> : 2개 이상의 <col> 묶어서 사용
- <col> : 열 1개에 지정 사용, 표의 열 개수 만큼 넣어야 한다.
    

#### 이미지 삽입
1. 이미지 삽입 <img> 태그
```html
<img src="이미지 파일 경로" alt="대체용 텍스트">
```
- <img> : 웹 문서에 이미지를 삽입할 떄 가장 기본인 태그이다.
  - src 속성 : 
    - 이미지 파일의 경로를 지정
    - 같은 폴더에 있다면 이미지 파일 이름만 적용 가능
    - "../" : 경로를 뒤로 이동 할때 사용
  - alt 속성 : 
    - 화면 낭독기 등에서 이미지를 대신해서 읽어 줄 텍스트를 입력
    - 인터넷인 불안정하거나 이미지 파일 경로를 잘못 넣었을 때 표시됨
  - width 속성 : 이미지의 너비를 조정
  - height 속성 : 이미지의 높이를 조정

  
##### 이미지 종류
  - GIF (Graphics Interchange Format)
    - **사용**: 주로 간단한 애니메이션과 로우 컬러 이미지에 사용됩니다.
    - **색상**: 최대 256색을 지원합니다.
    - **투명도**: 간단한 투명도 지원(완전히 투명하거나 완전히 불투명).
    - **압축**: 손실 없는 압축을 사용합니다.
  - JPG/JPEG (Joint Photographic Experts Group)
    - **사용**: 사진 및 현실적인 이미지에 주로 사용됩니다.
    - **색상**: 높은 색상 깊이를 지원하여 사실적인 이미지를 처리할 수 있습니다.
    - **투명도**: 투명도를 지원하지 않습니다.
    - **압축**: 손실 압축을 사용합니다. 압축률이 높을수록 이미지 품질이 저하될 수 있습니다.
  - PNG (Portable Network Graphics)
    - **사용**: 웹 이미지에 널리 사용되며, 투명도와 고품질 이미지를 필요로 하는 경우에 적합합니다.
    - **색상**: 다양한 색상 모드를 지원하며, GIF보다 더 많은 색상을 표현할 수 있습니다.
    - **투명도**: 알파 채널을 통한 투명도 지원으로, 이미지의 특정 부분을 다양한 수준의 투명도로 표현할 수 있습니다.
    - **압축**: 손실 없는 압축을 사용합니다.
##### px (픽셀)
- **정의**: 디지털 이미지의 기본 단위로, 스크린에 표시되는 이미지의 가장 작은 점.
- **사용**: 이미지, 요소, 레이아웃의 크기를 정확하게 지정하는 데 사용됨. 예: `width: 200px`.
- **특징**:
  - 절대적인 크기 단위, 사용자의 디스플레이 해상도나 브라우저 확대/축소에 영향을 받지 않음.
  - 고정된 크기로 인해 반응형 디자인에서 유연성이 떨어질 수 있음.

## % (퍼센트)
- **정의**: 백분율 단위, 상위 요소(부모 요소)의 크기에 대한 상대적인 비율로 사용됨.
- **사용**: 요소의 크기를 상위 요소의 크기에 비례하게 조정할 때 사용됨. 예: `width: 50%`.
- **특징**:
  - 상대적 크기 단위, 요소의 크기가 부모 요소의 크기에 따라 변함.
  - 반응형 웹 디자인에 적합, 다양한 화면 크기와 장치에 유연하게 적응함.

#### 오디오와 비디오 삽입
1. 다양한 멀티미디어 파일 삽입 <object>, <embed> 태그
```html
<object width="너비" height="높이" data="파일"></object>
```
- <object> : 오디오, 비디오, 자바애플릿, PDF 등 다양한 멀티미디어 파일을 삽입가능
  - width : 너비를 설정
  - height : 높이를 설정
```html
<embed src="파일경로" width="너비" height="높이">
```
- <embed> : 오디오, 비디오, 이미지 등 다양한 멀티미니더 파일을 삽입할 수 있다. <audio>, <video>, <object>가 지원하지 않는 웹 브라우저에서 사용
  
2. 멀티미디어 파일의 종류
- MP4
  - 다양한 멀티미디어 콘텐츠(비디오, 오디오, 자막, 2D 및 3D 그래픽)를 지원.
  - 고품질 비디오를 낮은 비트레이트로 제공, 인터넷 스트리밍에 적합.
  - 광범위한 장치 및 플랫폼에서 널리 지원됨.
- WebM
  - 고품질의 비디오를 웹에서 스트리밍하기 위해 개발됨.
  - VP8 또는 VP9 비디오 코덱과 Vorbis 또는 Opus 오디오 코덱을 사용.
  - 오픈소스이며, HTML5 비디오 표준의 일부로 지원됨.
  
- MP3
  - 손실 압축 오디오 형식으로, 음악 파일의 크기를 상당히 줄임.
  - CD 품질의 오디오를 제공하면서도 파일 크기를 작게 유지.
  - 가장 널리 사용되는 디지털 오디오 형식 중 하나.
- M4A
  - MPEG-4 Part 14 표준의 일부로 오디오 파일에 사용됨.
  - AAC 또는 ALAC 오디오 코덱을 사용하여 고품질 오디오 제공.
  - MP3에 비해 더 나은 음질을 가지면서 파일 크기는 비슷하거나 작음.
  
3. 오디오와 비디오 파일 삽입 <audio>, <video>
```html
<audio src="오디오 파일 경로" ></audio>
<video src="비디오 파일 경로" ></video>
```
- <audio> : 오디오를 삽입함
- <video> : 비디오를 삽입함

- 속성
  - controls : 컨트롤 바를 표시
  - autoplay : 자동 실행
  - loop : 반복 재생
  - muted : 소리를 제거
  - preload : 파일을 로딩할 것을 지정("auto(기본값)","metadata","none")
  - width, height : 너비와 높이를 지정
  - poster="파일이름"
